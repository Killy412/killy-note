# GeoHash

### GeoHash将经纬度转换为字符串,在大多数情况下,字符串前缀匹配的越多距离越近,

- Latitude范围 -90~+90
- longitude范围  -180~+180
- 地球参考球体的周长：40075016.68米,极半径：6356908.8米，赤道半径:6377830，平均半径：6371393
